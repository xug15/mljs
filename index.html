Welcome to exSEEK web service

exSEEK web service starts from a normalized data matrix of gene expression (read counts of each gene in each sample) and performs feature selection, evaluation, prediction, which also provide the panel marker information and clinical information.

The exSEEK web service is built on an entirely serverless architecture. Using machine learning methods based on JavaScript, this design yields a significant performance increase at a lower cost compared to a traditional architecture.

If you think you‘ve found a bug, the best way to get in touch is to make an bug report on the project’s GitHub repository. To make a feature request for the exSEEK web server, please click here.
